Using Numbers
Let's see how we can do this without using pictures or diagrams – just numbers.
The first thing to notice is that we can have a maximum of three children with every adult. So we can try to fill the coach with groups of four (three children and one adult in each group).
So, how many of these groups of four can we fit on the coach? We need to divide...
How many fours go into 27? Six fours make 24, so four into 27 goes six times, with a remainder of three.
Each group of four includes three children, so that's 6 × 3 = 18 children, and three seats left. Let's think about what we can do with those three seats...
There are already three children to every adult – so we need to add another adult. We can then put children into the remaining two seats, so the total number of children who can go is 18 + 2 = 20.
